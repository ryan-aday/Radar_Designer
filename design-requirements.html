Capture the essentials before sizing hardware: power, antenna gain, target radar cross section (RCS), range, speed, and environment. Each item links to governing equations pulled from the eGuide.

Mission checklist

Detection goals

  • Maximum instrumented range, R
  • Probability of detection PD (typ. 0.8–0.9)
  • False alarm rate FAR and CFAR strategy
  • Range & angular resolution needs (cell size ΔR, Δθ)

Target set

  • Expected RCS range (see RCS table on Equations page)
  • Radial speeds and Doppler constraints
  • Radar horizon / height-of-target considerations

Environment

  • Frequency band and ISM constraints (Fig 1)
  • Atmospheric / rain attenuation (Fig 10)
  • Ground effects: Fresnel clearance and earth bulge (Figs 8–9)

EW posture

  • Jamming threat and desired burn-through range
  • Waveform agility (PRF jitter, pulse compression, LPI goals)
  • Polarization, sidelobe control, and ECCM features

Quick-start equations

Two-way radar equation (monostatic)

PRX = (PTX·G2·λ2·σ) / ((4π)3·R4·Ltot)

Use the loss table (page 45) to accumulate Ltot. Square-root dependency replaces fourth-root for volume weather targets.

Free-space path loss

FSPL = 20 log10(4πR/λ)

Switch to the two-ray model when the link spans beyond the Fresnel distance or grazes the ground (Fig 12).

Burn-through estimate

RBT ≈ √( (PTX·GM) / (4π·J) )·(λ/ (4π))

Compare to the support-jamming equation on the EW page for escort/sidelobe cases.

Range, speed, and waveform dependencies

Resolution

Range resolution: ΔR = c0·τ/2 (or compressed pulse width). Angular resolution depends on half-power beamwidth θ; targets spacing > θ/2 is separable.

Doppler window

Max unambiguous velocity: vu = λ·PRF/4. Increase PRF for Doppler coverage but trade unambiguous range (Doppler dilemma).

Duty cycle

Duty factor = τ / PRT. Pulse compression enables long energy-on-target with shorter effective blind ranges.
